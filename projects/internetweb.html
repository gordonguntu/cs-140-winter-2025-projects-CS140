Internet

It is a worldwide system of millions of networks connected to computers and other electronic devices that help communicate with each other.

Web

Web refers to all the websites that people can use to access on their electronics through internet.

How are the Internet and Web different?

Internet is a global network that connects to other networks while web is a service that runs on the internet.

ASCII and Unicode

ASCII and Unicode are two encoding schemes, ASCII encodes symbols and letters and Unicode encodes special texts from different symbols and letters.

What is UTF-8

UTF-8 is a character coding system on the web which represents characters using one to four bytes.

URL

URL stands for Uniform Resource Locator, it is used to locate resources on the internet.

Domain Name

It is a name of a website address which indentifies that it belongs to a particular domain.

HTTP

It stands for Hypertext Transfer Protocol is an application protocol that allows users to communicate data on world wide web.

HTTPS

HTTPS stands for Hypertext Transfer Protocol Secure and is a secure versi0on of HTTP, it is encrypted so that the data transfer will be secure.

IP Address

It is a unique identifying number given to all the devcices that are connected to the internet.

DNS

DNS stands for Domain Name System, It is a naming database where all the domain names are located and translated into IP address.

W3C Recommendations

W3C Recommendations is a set of rules and specifications that creates standard for world wide web.

XML, XHTML, HTML, HTML5

They are markup languages that indicates logical structure and gives instructions for laying out a document on a page for display.
